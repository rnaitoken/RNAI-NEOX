# 🧠 RNAI-NEOX Genesis Pro

**RNAI-NEOX** is a modular smart automation system powered by AI, designed to integrate with multiple APIs, manage autonomous bots, and scale digitally using no-code technology.

> 🌌 This is the flagship project of the RNAI Genesis community: bots that think, execute, and replicate to fulfill digital missions.

---

## 🌟 Features

- 🎛️ Galactic control panel (`index.html`) with API search
- 🤖 Easily connect APIs using smart no-code forms
- 🧩 Modular folder structure for each integration (`openai/`, `telegram/`, etc.)
- 🗂️ Independent dashboards for each API
- 📦 Active integrations:
  - OpenAI
  - Telegram Bot
  - Zoom API
  - Zapier Webhooks
  - ElevenLabs
  - Google Cloud
  - Discord Bots
  - WhatsApp Cloud API
  - Notion API
  - HuggingFace
